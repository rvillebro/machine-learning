Activation functions (activation_functions)¶

This module contains all available activation function for layers: ReLU, bla and bla. All activation functions must be a subclass of ActivationFunction.

class activation_functions.ActivationFunction[source]¶

Superclass for all activation functions. Used for defining which methods must be implemented when implementing a new activation function.

class activation_functions.ReLU[source]¶

Rectified Linear Unit (ReLU) activation function (superclass: ActivationFunction)

\[ \begin{align}\begin{aligned}ReLU(x) = x if x > 0 otherwise 0\\ReLU'(x) = 1 if x > 0 otherwise 0\end{aligned}\end{align} \]

ReLU(x) = x if x > 0 otherwise 1

ReLU’(x) = 1 if x > 0 otherwise 0
